# Installation

Welcome to Orion! Follow these steps to install and set up your environment.

## System Requirements
- **Windows:** Windows 10 or later (64-bit)
- **macOS:** macOS 10.15 or later
- **Linux:** Ubuntu 20.04 or later (requires manual compilation)

## Installation Steps
1. Download the latest installer from our website.
2. Run the installer and follow the on-screen instructions.
3. Launch Orion.
4. On first launch, the Setup Wizard will guide you through selecting your User Data Folder.

## Audio Setup
1. Go to **Settings > Audio**.
2. Select your audio device driver (ASIO recommended on Windows).
3. Choose your input and output channels.
4. Set your sample rate and buffer size (lower buffer size = lower latency, but higher CPU usage).

## MIDI Setup
1. Go to **Settings > MIDI**.
2. Enable your MIDI inputs and outputs.
3. Use the "Test" button to verify MIDI connectivity.

You are now ready to start making music!
