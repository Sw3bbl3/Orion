# Basic Usage

Orion is designed to be intuitive. Here's how to navigate the interface.

## The Dashboard
The Dashboard is your starting point. Here you can:
- Open recent projects.
- Start a new project.
- Access settings and documentation.
- Manage extensions.

## Main Interface
Once a project is open, you will see the main workspace:
- **Top Bar:** Transport controls (Play, Stop, Record), tempo, and time signature.
- **Left Panel:** Browser for files, plugins, and samples.
- **Center:** The Arrangement View (Timeline) where you arrange your clips.
- **Bottom:** The Mixer or Editor view.

## Creating Your First Track
1. Drag an instrument from the Browser to the Timeline.
2. A new track will be created automatically.
3. Double-click on the timeline to create a MIDI clip.
4. Double-click the clip to open the Piano Roll.
5. Draw notes to create a melody.

## Adding Effects
1. Select a track in the Mixer.
2. Drag an effect from the Browser to an empty slot in the Insert Rack.
3. Adjust parameters to shape your sound.

## Saving
Use `Ctrl+S` (Windows) or `Cmd+S` (macOS) to save your project.
Projects are saved as `.orion` files.
