# The Dashboard

The Dashboard is the central hub of Orion. It appears when you launch the application or close a project.

## Sections

### Home
Shows your most recent projects for quick access. You can pin projects to keep them at the top.

### Favorites
Lists projects you have marked as favorites. Click the star icon on any project card to favorite it.

### All Projects
A comprehensive list of all projects found in your User Data Folder. Use the search bar to find specific projects.

### Learn
Access documentation, tutorials, and guides (like this one!).

### Extensions (Developer Mode)
Manage installed extensions and create new ones. This section is only visible if Developer Mode is enabled in Settings.

### Settings
Configure Audio, MIDI, General preferences, and manage your User Data Folder location.

## Project Cards
Each project card displays:
- Project Name
- Tempo (BPM)
- Last Modified Date
- A waveform preview (if available)

Hover over a card to play a preview of the project audio.
Right-click a card for more options:
- Open Project
- Show in Explorer/Finder
- Add/Remove from Favorites
- Delete Project
