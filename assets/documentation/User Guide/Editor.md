# The Editor

The Editor panel adapts based on the selected content.

## Piano Roll (MIDI)
When you select a MIDI clip, the Piano Roll appears.
- **Grid:** The background grid represents musical time (bars and beats).
- **Notes:** Draw notes to create melodies and chords.
    - **Click:** Add a note.
    - **Click + Drag:** Resize a note.
    - **Right-Click:** Delete a note.
- **Velocity:** The bottom lane controls the velocity (loudness) of each note.

## Audio Editor
When you select an audio clip, the Audio Editor appears.
- **Waveform:** Visualizes the audio content.
- **Fade Handles:** Drag the corners of the clip to create fade-ins and fade-outs.
- **Gain:** Adjust the clip gain using the center handle.
- **Time Stretching:** Hold `Alt` and drag the edge of a clip to time-stretch it.

## Automation
Switch to Automation Mode to draw curves for parameters (Volume, Pan, Plugin Parameters).
- Click to add points.
- Drag lines to curve them.
