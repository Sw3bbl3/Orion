# Keyboard Shortcuts

Mastering keyboard shortcuts is the fastest way to improve your workflow in Orion.

## Global Transport

| Action | Shortcut |
|---|---|
| Play / Stop | `Space` |
| Record | `R` |
| Loop Toggle | `L` |
| Fast Forward | `.` |
| Rewind | `,` |
| Return to Zero | `Return` |

---

## Arrangement Tools

| Action | Shortcut |
|---|---|
| Select Tool | `1` |
| Blade Tool | `2` |
| Glue Tool | `3` |
| Erase Tool | `4` |
| Select All | `Cmd/Ctrl + A` |
| Split at Playhead | `Cmd/Ctrl + B` |

---

## Views & Windows

| Action | Shortcut |
|---|---|
| Toggle Mixer | `M` |
| Toggle Inspector | `I` |
| Toggle Browser | `B` |
| Toggle Piano Roll | `P` |
| Command Palette | `Cmd/Ctrl + Shift + P` |

---

## Editing & Navigation

| Action | Shortcut |
|---|---|
| Undo | `Cmd/Ctrl + Z` |
| Redo | `Cmd/Ctrl + Shift + Z` |
| Duplicate | `Cmd/Ctrl + D` |
| Zoom In | `+` / `Mouse Wheel Up` |
| Zoom Out | `-` / `Mouse Wheel Down` |
| Nudge Left/Right | `Arrows` |

> [!TIP]
> You can customize these shortcuts in **Settings > Shortcuts**.
